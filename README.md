A set of VUE+IVIEW based templates.
一套基于VUE+IVIEW的模板。

# iView template
专注于模板外框架，方便程序员初始化界面。

--管理员模板
login
master
home

## 安装及使用
```bush
clone 克隆下来
npm install 安装依赖
npm run dev 编译生成
npm run start 启动后端服务器
url：http://localhost:8080  根据后端服务器的baseUrl访问网址，显示出app则表示正常
```
### 管理后台地址
```bush
http://localhost:8080/admin/login 登录页面
http://localhost:8080/admin/home  管理首页
```
